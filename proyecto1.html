Descripción del Proyecto

Este proyecto consiste en una solución completa de e-commerce diseñada para permitir a pequeñas y medianas empresas (PyMEs) gestionar su inventario, pedidos y ventas en línea de manera eficiente. El desarrollo se centró en la escalabilidad y la facilidad de uso para administradores no técnicos.

Retos Superados:

El mayor desafío fue optimizar las consultas a la base de datos para el módulo de inventario.

Fragmento de Código Clave:


// Consulta optimizada para productos con cache
$products = Cache::remember('popular_products', 60*24, function () {
    return Product::where('is_popular', true)
        ->with('category')
        ->limit(10)
        ->get();
});
