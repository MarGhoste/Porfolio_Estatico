Descripción del Proyecto

Una plataforma de e-commerce moderna y completa construida con el stack VILT: Vue.js, Inertia.js, Laravel y Tailwind CSS. Este proyecto proporciona una base sólida para una tienda en línea, con un catálogo de productos, carrito de compras, autenticación de usuarios y un panel de administración.

⭐ Características Principales ⭐

  • Catálogo de Productos: Navegación por productos con imágenes, descripciones y precios.
  • Carrito de Compras: Funcionalidad completa para agregar, eliminar y actualizar productos.
  • Autenticación Segura: Registro e inicio de sesión para usuarios.
  • Panel de Administración: Gestión de productos, categorías y pedidos.
  • Pasarela de Pago: Integración con Stripe para procesar pagos de forma segura.

Este proyecto es un M.V.P. desarrollado en 8 días (≈60 horas).

Retos Técnicos Superados:

El principal desafío fue la integración de la Pasarela de Pago Stripe a través de Laravel Cashier, manejando flujos complejos del servidor para pagos seguros y la creación de órdenes.
