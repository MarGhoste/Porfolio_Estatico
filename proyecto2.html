Descripción del Proyecto

Plataforma de E-commerce Una plataforma de e-commerce moderna y completa construida con el frontend del stack VILT: Laravel, Inertia.js y Vue.js. Este proyecto proporciona una base sólida para una tienda en línea, con un catálogo de productos, carrito de compras, autenticación de usuarios y un panel de administración.

Acerca del Proyecto

Esta aplicación es una demostración de un flujo de trabajo de desarrollo web moderno, integrando un potente backend en PHP con un frontend reactivo de tipo Single-Page Application (SPA). Utiliza las mejores herramientas de su clase para proporcionar una experiencia fluida tanto para desarrolladores como para usuarios.


⭐ Características Principales ⭐

Catálogo de Productos:
Los usuarios pueden navegar por una lista de productos con imágenes, descripciones y precios.


Carrito de Compras:
Funcionalidad completa de carrito que permite a los usuarios agregar, eliminar y actualizar productos antes de proceder al pago.


Autenticación de Usuarios:
Registro e inicio de sesión seguros para una experiencia personalizada.


Panel de Administración:
Interfaz para que los administradores gestionen productos, categorías y pedidos.


Este proyecto es un M.V.P. (Producto Mínimo Viable) completo, desarrollado en tan solo 8 días con promedio de 5-8 horas al dia (aproximadamente 60 horas). Demuestra mi capacidad de construir una aplicación full-stack moderna y productiva.

Retos Superados:

El principal desafío fue la integración de la Pasarela de Pago Stripe a través de Laravel Cashier. Esto requirió gestionar flujos de trabajo complejos del lado del servidor para el manejo seguro de tokens de pago y la correcta sincronización de datos de la aplicación con los servicios de Stripe, garantizando una experiencia de pago 100% segura y conforme a la normativa de la industria..
