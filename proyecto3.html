Real-Time To-Do App

Una aplicación de lista de tareas con actualizaciones instantáneas usando Livewire y Echo.

Tecnologías Usadas:

Livewire Alpine.js Tailwind CSS
Ver Código (GitHub)

Descripción del Proyecto

Esta es una aplicación de lista de tareas (To-Do) que demuestra el poder de la reactividad en tiempo real del TALL Stack. Las tareas se actualizan instantáneamente para todos los usuarios conectados gracias a la combinación de Laravel, Livewire, Alpine.js y la difusión de eventos con Laravel Echo.

Retos Superados:

El principal reto fue configurar Laravel Echo y Pusher para transmitir eventos de manera eficiente y segura, asegurando que las actualizaciones de la UI fueran instantáneas y solo para los canales correctos.

Fragmento de Código Clave: 


// Disparar evento al crear una tarea
TaskCreated::dispatch($task);

// Escuchar evento en Livewire
#[On('echo:tasks,TaskCreated')]
public function onTaskCreated()
{
    $this->refreshTasks();
}
